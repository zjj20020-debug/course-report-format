# 课程报告格式

## ⚠️ 强制流程（不可跳过任何步骤）

### Step 1 — 创建 report_content.json

在工作目录下创建 `report_content.json`。复制 skill 目录的 `report_content.json` 作为模板，填入内容。

**Json 字段说明：**

| 字段 | 必填 | 说明 |
|------|------|------|
| `title` | ✅ | 中文题目，二号黑体不超过20字 |
| `title_en` | | 英文题目，4号TNR加粗 |
| `author` | | 作者，默认"作者姓名" |
| `author_en` | | 英文作者，姓氏全大写 |
| `affiliation` | | 单位 |
| `affiliation_en` | | 英文单位 |
| `abstract` | ✅ | 中文摘要约200字 |
| `abstract_en` | | 英文摘要 |
| `keywords` | ✅ | 分号分隔，8~10个词 |
| `keywords_en` | | 英文关键词 |

**Section 结构：**
- `heading` — 一级标题（1, 2, 3…）
- `subsections[].subheading` — 二级标题（1.1, 2.1…）
- `paragraphs` — 字符串数组。可混入 `{"type":"table",...}` 或 `{"type":"figure",...}` 对象
- `refs` — 参考文献数组，英文用 `{"type":"en","text":"[1] ..."}` ，中文用 `{"type":"cn","cn":"[2] 中文...","en":"ENGLISH..."}`

**公式写法：**
- 行内：`$\\Gamma_n = \\beta_n \\cdot \\exp(j\\varphi_n)$`
- 独立：`$$\\Phi = \\mathrm{diag}(...)$$`
- JSON 中反斜杠**必须双写**

**引用规范：**
- 中文引号「」替代 ""
- 不用 `\n`，段落拆分为数组不同元素

### Step 2 — 检查依赖

```bash
pip install lxml python-docx   # Python 依赖
pandoc --version                # 确认 pandoc >= 3.x
```

### Step 3 — 运行脚本

```bash
python ~/.claude/skills/course-report-format/generate_report.py report_content.json
```

脚本会：
1. pandoc --mathml 渲染所有 `$...$`/`$$...$$` 公式为 Word 原生 OMML
2. 生成中英双语题头
3. 渲染三线表 + 中英双语图注
4. 追加参考文献（英TNR/中宋体，固定行距12磅）
5. 输出 `RIS_6G_Report.docx`

### Step 4 — 验证

确认输出文件存在、大小合理（> 10 KB）、在 Word 中能正常打开且无报错。

## 格式规范参考

### 页面
A4，上 2.6cm 下 1.6cm 左 2.0cm 右 2.0cm。题头单栏 → 正文双栏（continuous 不分页）。

### 字体字号

| 元素 | 字体 | 字号 | 格式 |
|------|------|------|------|
| 中文题目 | 黑体 | 二号 22pt | 居中，段前60磅段后15磅，单倍行距 |
| 中文作者 | 楷体 | 小四 12pt | 居中，固定行距15磅，段后6磅 |
| 中文单位 | 楷体 | 小五 9pt | 居中，固定行距15磅 |
| 「摘  要：」标签 | 黑体加粗 | 小五 9pt | — |
| 摘要内容 | 楷体 | 小五 9pt | 固定行距15磅，左右缩进2字符 |
| 「关键词：」标签 | 黑体加粗 | 小五 9pt | — |
| 关键词内容 | 宋体 | 小五 9pt | 固定行距15磅 |
| 英文题目 | TNR 加粗 | 四号 14pt | 居中，单倍行距，段后15磅 |
| 英文作者 | TNR 正体 | 小四 12pt | 居中，固定行距15磅 |
| 英文单位 | TNR | 小五 9pt | 居中，段后6磅 |
| Abstract 标签 | TNR 加粗 | 小五 9pt | — |
| Abstract 内容 | TNR | 小五 9pt | 左右缩进2字符，段后15磅 |
| Keywords 标签 | TNR 加粗 | 小五 9pt | — |
| Keywords 内容 | TNR | 小五 9pt | 段后12磅 |
| **正文** | **宋体** | **五号 10.5pt** | **首行缩进2字符，单倍行距，两端对齐** |
| **一级标题** | **仿宋加粗** | **四号 14pt** | **固定行距15磅，段前段后10磅** |
| **二级标题** | **黑体加粗** | **五号 10.5pt** | **固定行距15磅，段前段后3磅** |
| 参考文献标题 | 黑体加粗 | 四号 14pt | 单倍行距 |
| 参考文献正文 | 英TNR/中宋体 | 小五 9pt | 固定行距12磅 |

### 表格
三线表（顶线底线 1pt，栏目线 0.75pt，无竖线），表内小五宋体，表序表题小五黑体中英双语，表宽 ≤ 7.4cm（适配双栏）。

### 图片
小五号字，图序图题小五黑体中英双语，宽 ≤ 7.4cm，尽量不用彩色。

### 参考文献
- 作者姓氏在前全大写，名在后首字母大写
- 3人以上列出前3人，加"等"/"et al."
- 中文文献下方附英文翻译（不编号）
- 建议 ≥ 10条

## 常见错误排查

| 症状 | 原因 | 解决 |
|------|------|------|
| JSON 解析报错 | 中文引号""冲突 | 改为「」 |
| 公式不渲染 | 反斜杠未双写 | `\\Gamma` 而非 `\Gamma` |
| Word 打不开 | XML 损坏 | 检查 Content_Types.xml 是否有重复条目 |
| pandoc 报错 | 版本 < 3.x | `winget install pandoc` |
